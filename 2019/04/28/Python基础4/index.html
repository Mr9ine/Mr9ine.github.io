Python学习笔记(函数&类&实例）

#（接）函数
python中有不同于之前所学的新东西——模块。经过学习可知，模块是一类独立文件，用import语句可以允许在当前运行的程序文件中使用模块中的代码

##tips：

  1. 模块扩展名是.py，可以导入整个模块
  2. 也可以导入特定的函数，如：
    from module import function_0,function_1,function_2
  3. 使用as给函数指定别名，如
    from module import function_0 as fc_0
  4. 还可以用as给模块指定别名，如
    import food as f
  5. 导入模块中的所有函数，如
    from food import *
    函数命名中，使用小写字母和下划线，切给形参指定默认值时候，等号两边不要有空格，调用时也要遵循这种约定
